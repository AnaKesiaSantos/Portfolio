GESTÃO DE ENERGIA E ÁGUA EM CASAS INTELIGENTES


Este projeto, desenvolvido como parte do curso de Análise e Desenvolvimento de Sistemas, teve como objetivo criar um sistema de gestão inteligente de energia e água em residências. Utilizando conceitos de Internet das Coisas (IoT), foi implementado um conjunto de dispositivos e um aplicativo que permite o monitoramento em tempo real do consumo de energia e água, identificando desperdícios e promovendo a eficiência no uso desses recursos. O sistema propõe soluções inovadoras, como tomadas, disjuntores e medidores de vazão inteligentes, que otimizam o consumo e contribuem para uma vida mais sustentável, reduzindo custos e impactos ambientais. Gestão de Casa de Repouso - Aplicação Computacional em Java

Este projeto foi desenvolvido como parte da disciplina Laboratório de Programação II, com o objetivo de aplicar os conhecimentos adquiridos sobre manipulação de arquivos e desenvolvimento de aplicações em Java. O tema sorteado foi "Casa de Repouso", e a proposta incluiu a criação de uma aplicação que gerencie informações relevantes para esse ambiente, utilizando armazenamento de dados em arquivos de texto (TXT). A aplicação foi projetada para funcionar com dois arquivos de dados inter-relacionados, permitindo o registro e gerenciamento de informações dos residentes e das atividades realizadas. O sistema oferece um menu de navegação com as seguintes funcionalidades: Inserção de Dados: Permite adicionar informações nos dois arquivos (de forma separada ou conjunta). Alteração e Exclusão de Dados: Funcionalidade para modificar e remover dados da segunda tabela. Consulta Geral e Específica: Exibe informações de ambos os arquivos e permite realizar pesquisas com base em um dado específico fornecido pelo usuário. Interface Interativa: O sistema foi desenvolvido com uma interface simples e amigável, garantindo a consistência dos dados e apresentando mensagens informativas e de erro para o usuário, quando necessário. Site Informativo - Setembro Amarelo


Este projeto foi desenvolvido como parte dos requisitos da disciplina de Tópicos especiais, com o objetivo de criar um site navegável de no mínimo
cinco páginas, sem a utilização de frameworks. O tema escolhido para o site foi Setembro Amarelo, uma campanha nacional de prevenção ao suicídio e
promoção da saúde mental. O site foi construído utilizando apenas HTML e JavaScript, e todas as páginas são interligadas por meio de um menu de navegação,
permitindo uma experiência fluida para o usuário. As páginas abordam tópicos como a introdução ao Setembro Amarelo, explicando seu histórico e objetivo, sinais
de alerta para identificar pessoas em risco, além de oferecer informações sobre onde buscar ajuda com contatos de profissionais e organizações de apoio. Também há
uma página com depoimentos e relatos que reforçam 7a importância da conscientização sobre saúde mental. Para a elaboração do design das páginas, foi sugerida a
adaptação de exemplos do framework Bootstrap, porém, o desafio foi desenvolver o projeto sem a utilização de frameworks. O design final foi adaptado para manter uma estrutura
organizada e amigável, seguindo boas práticas de usabilidade e acessibilidade, resultando em um site simples, mas informativo, focado em transmitir de maneira clara a mensagem de apoio e conscientização sobre o Setembro Amarelo. Veja no GitHub
